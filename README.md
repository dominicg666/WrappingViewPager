# WrappingViewPager
Android Library for wrap content ViewPager

 #gradle
  Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
 Add the dependency

	dependencies {
		implementation 'com.github.dominicg666:WrappingViewPager:1.0'
	}
